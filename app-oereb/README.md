# app-oereb (legacy redirect)

This folder used to hold the ÖREB Parcel Search app. It moved to [`../oereb-search/`](../oereb-search/) during the May 2026 repo cleanup.

The [`index.html`](index.html) here is a tiny redirect stub kept so old bookmarks to `https://bbl-dres.github.io/geo-check/app-oereb/` keep working — it sends visitors to `…/oereb-search/`.

**Do not add new files here.** Add them to [`../oereb-search/`](../oereb-search/) instead.

This stub can be deleted once the redirect is no longer needed (e.g. once analytics show no traffic on the old path for a few months).
